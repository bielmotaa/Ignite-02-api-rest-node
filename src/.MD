## Rodar TypeScript sem compilar manualmente

Em vez de usar `npx tsc` toda vez para gerar o `.js`, do arquivo que eu criei em js, e só depois rodar o arquivo, dá para usar o **tsx** e executar direto:

### Como configurar

```bash
npm install tsx -D
```

### Como usar

```bash
npx tsx src/server.ts
```

O `tsx` compila o TypeScript na hora e roda o arquivo em um único comando. - isso eh bom no desenvolvimento, em producao devemos converter.
